What is this project about?

>>Please use Google Chrome browser for the experience using this web app!<<
Nina Mir | CS graduate student | San Francisco State University | May 2020

In short, using the San Francisco Police Department Incident Reports: 2018 to Present dataset available here, I created a web app for an interactive data visualization system, consisted of two views: & place views, to offer a more clear picture of the nature, numbers and geo-location of all the incident calls dated 2019.

System Architecture

  • Operating System: Linux Ubuntu 16.04
  • Server Environment: Node.js (version: 8.17.0)
  • Application Framework: Express.js (version: 4.17.1)
  • Frontend: jQuery, JavaScript, Bootstrap, CSS, HTML
  • Data Visualization Libraries: Google Chart API, D3 library (d3.v4.min.js)
  • Data Cleaning: Python Pandas

Dataset Description

The original dataset (~ 120 MB) can be downloaded in a CSV format containing 344,000 rows and 26 columns. Each row contains numerous temporal, categorical, and geographical information pertaining to one incident call: incident datetime, day of the week, incident category, incident subcategory, incident description, incident resolution, neighborhood, police district, supervisor district, latitude, longitude, nearest street intersection, etc.  An example row is as follows:
